Template Jest Tailwind TypeScript

Este template esta ideado para el uso de Tailwind con TypeScript.

Test:
Incluye la biblioteca Jest para realizar test.

Refresco:
Al ejecutar `npm run build` se refresca automáticamente el output del css y el ts.
Al ejecutar `npm run start:server` se ejecuta el servidor, detectando cualquier cambio en los archivos de este.
Para ver los cambios en los archivos estáticos (HTML y CSS) es necesario refrescar la pagina o utilizar otras herramientas como nodemon y cambiar la configuración.
